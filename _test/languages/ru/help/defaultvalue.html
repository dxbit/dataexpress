Значение по умолчанию записывается в поле при добавлении новой записи. Является выражением. В случае установки значения по-умолчанию для объекта, следует помнить, что объект хранит идентификатор записи (т. е. число). Выражение может иметь следующий вид:
objid('форма', 'поле', 'значение')
objid('Сотрудники', 'Логин', user)
objid('Склады', 'Название', 'Склад № 1')

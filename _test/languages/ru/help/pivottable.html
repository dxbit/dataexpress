Сводная таблица позволяет отобразить данные в более удобном для восприятия виде, может группировать данные и рассчитывать промежуточные и общие итоги. Сводная таблица имеет три измерения: строки, столбцы и данные. Чтобы создать сводную таблицу нужно как минимум указать по одному полю для каждого измерения. Поля строк фомируют заголовки по вертикали, поля столбцов - заголовки по горизонтали. Данные в строках и столбцах упорядочиваются по возрастанию. На пересечении строк и столбцов находятся поля данных.

Программа имеет широкие возможности по настройке внешнего вида таблицы. Настраивается буквально каждое поле. Сводная таблица имеет следующие функции вычисления итогов: сумма, среднее, максимум, минимум и количество.

Таблица берет данные из запроса. При настройке выбирается запрос из текущей формы. Сам запрос можно скрыть под таблицей, она дублирует его свойства.

Тип данных поля. Каждое поле в сводной таблице имеет тип данных. Это необходимо для правильной сортировки и вычислений. При выборе поля программа пытается самостоятельно определить его тип данных. Но часто его нужно корректировать (особенно это касается вычисляемых полей, тип данных которых текстовый). Тип данных настраивается в свойствах поля в закладке "Итоги". Всего 4 типа данных: текст, число, дата, время.

Расчет итогов. Для правильного вычисления итога нужно указать правильный тип данных. При расчете итога пустые ячейки игнорируются. Также игнорируются ячейки, которые не соответствуют типу данных (например, вместо числа в ячейке текст). Программа может рассчитывать промежуточные итоги для полей строк и полей столбцов. Скрыть/показать промежуточный итог можно в свойствах поля в закладке "Итоги".

Цвет. Цвет можно настроить для каждого поля. Настройки цвета одного поля могут перекрываться другим из другого измерения. Наивысший приоритет у полей данных, затем в порядке убывания: поля строк, поля столбцов, цвет общих итогов, цвет области данных. Если цвет имеет значение None, то он игнорируется и используются настройки с более низким приоритетом.

Объединение данных. Возможна ситуация, когда на пересечении строки и столбца может быть несколько значений. В этом случае они объединяются в одну строку. Значения разделяются символами, которые указываются в свойстве "Разделитель данных".

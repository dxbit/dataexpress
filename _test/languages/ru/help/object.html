Объект - ключевая фигура в базе данных. Это поле создает связь между двумя формами: формой, на которой объект расположен и формой, которая выбрана в качестве источника списка. Эта связь заключается в том, что вы можете обращаться к данным связанной формы через объект. Поле хранит идентификатор объекта, ссылку на запись связанной формы. Любые изменения в связанной форме отражаются на объекте. Нельзя удалить запись в связанной форме, если на нее ссылается объект.

В выражениях и шаблонах, чтобы обратиться к полю объекта (данным связанной формы), надо писать так:
[Объект|Поле объекта]
[Сотрудника|ФИО]
[Товар|Единица измерения|Название]

Сначала записывается имя объекта, затем имя поля связанной формы. Поле объекта тоже может быть объектом. Глубина вложения объектов не ограничена.

Если щелкнуть мышкой на поле с кнопкой Ctrl, то программа переместит к связанной записи. Это работает в любом представлении формы, кроме представления Только таблица.

Поле имеет кнопку, которая открывает связанную форму. В отличии от списка, который загружается только один раз при попытке открыть список или вводе текста, данные связанной формы обновляются всегда. В контекстном меню кнопки есть команды для добавления и редактирования связанных записей.

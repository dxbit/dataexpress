В качестве источника отбора может выступать форма и одна подчиненная таблица. Таблица указываеся при необходимости. Выберите поля, которые вы хотите видеть в результате. Для этого нажмите кнопку "Добавить поле" и в новой строке выберите поле. По некоторым полям можно посчитать итог. Для этого напротив поля, в колонке "Фукнция" выберите нужную функцию. Назначение функций:

  • Сумма - вычисляет итоговую сумму.
  • Среднее - вычисляет среднее значение.
  • Максимум - вычисляет максимальное значение.
  • Минимум - вычисляет минимальное значение.
  • Количество - считает число записей.
  • Баланс - вычисляет разницу прихода и расхода. Нужно обязательно указывать тип источника: приход или расход.
  • Кол-во различных - считает кол-во уникальных значений поля.
  • Соединить - соединяет значения поля в одну строку, разделяя значения точкой с запятой и пробелом. В результат не включаются повторяющиеся значения.
  • Соединить все - соединяет значения поля в одну строку, разделяя значения точкой с запятой и пробелом. В результат входят все значения, в том числе повторы.

Для функций Сумма, Среднее, Баланс подходят только числовые поля. Для остальных функций подходят любые поля.

Для вычисления функции необязательно указывать поля для всех источников. Для функций "Сумма" и "Баланс" если в каком-либо источнике поле не указано, то будет подставляться 0. Это нужно для правильного вычисления итогов. Например, если вам нужно расчитать приход, расход и баланс в разных колонках, для прихода выбирается функция "Сумма", и поля выбираются только для источников с типом "Приход", а в источниках типа "Расход" ничего не выбирается - будет подставлен 0. То же самое и для расхода. Если выбрана функция "Среднее", то подставляется NULL. NULL не учитывается при расчете среднего. Для функции "Количество" если не выбрано ни одного поля, то считается общее количество записей. Если выбрано хотя бы одно поле, то считаются все не-NULL значения в этом поле.

Для отбора можно указать более одного источника. Данные источников будут объединены. Типы полей разных источников должны совпадать или быть совместимыми. Допускается не указывать поля для какого-либо источника. В этом случае в ячейку будет записано значение:
  • 0 - если выбрана функция "Сумма", "Баланс";
  • 0 - для функции "Количество", если не выбрано ни одного поля, NULL - если выбрано хотя бы одно поле;
  • пустая строка - если выбрана функция "Соединить", "Соединить все";
  • NULL - в остальных случаях.

Данные могут группироваться по дате. Введите имя поля (колонка "Имя поля") и выберите период: день, неделя, месяц, квартал, полугодие, год.

При просмотре отчета можно использовать фильтр. Для этого поставьте галочку в колонке "Параметр" напротив того поля, по которому хотите фильтровать. Если это поле не должно отображаться в результате, то снимите галочку "Видимое".

По каждому источнику можно задать условие отбора в строке "Фильтр". Если отбор настраивается для запроса, то в выражении можно указывать поля формы. В этом случае данные полей формы будут подставляться в фильтр (будут действовать как параметры).

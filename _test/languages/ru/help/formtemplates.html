Готовые шаблоны должны хранится в папке для шаблонов. По умолчанию, это папка templates. Она находится в папке программы. Вы можете указать другую папку в настройках программы.

Каждый шаблон указывается в отдельной строке. Должны быть указаны имя файла и расширение. Допустимо указывать относительный путь к файлу шаблона, т. е. в папке шаблонов могут быть подпапки. При вводе данных кнопка Печать станет доступной. Эта кнопка будет по прежнему недоступна, если форма не содержит записей. Если указано несколько шаблонов, то при нажатии на кнопку будет открываться меню, в котором выбирается шаблон. Если указан один, то печать будет произведена немедленно.

Более подробную информацию вы найдете в разделе Шаблоны.

Хранит значение счетчика. При добавлении новой записи поле остается пустым. При сохранении записи программа проверяет: если поле пустое, то в него записывается значение счетчика, увеличенное на единицу. Если поле было заполнено, то значение счетчика не записывается и не увеличивается. Можно и вовсе запретить редактирование этого поля.

По умолчанию, начальное значение счетчика равно 0, но вы можете его изменить в любое время.

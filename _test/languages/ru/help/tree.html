Дерево может отображать значения некоторого объекта формы, например групп товаров/услуг. В режиме ввода слева от табличной части формы появляется древовидный список. При выборе значения из списка данные фильтруются по этому значению. Допускается выбирать несколько значений. Для этого выбирайте значение с нажатой клавишей Ctrl. Дерево и фильтр формы синхронизируются. Т. е. выбранные в дереве значения добавляются в фильтр формы. И напротив, при выборе значений объекта в фильтре, в дереве эти значения выделяются. При вставке новой записи, значение выбранной группы будет подставляться в соответствующее поле.

В окне настройки свойства "Дерево" необходимо выбрать поле-обект, значения которого будут отображаться в дереве. Допускается использование любых объектов. Также в этом окне настраивается внешний вид дерева.

Это свойство позволяет выбрать форму и поле, значениями которого будет заполнен список. Для объекта это свойство обязательно, т. к. оно создает связь между объектом и формой. В выражениях и шаблонах вы можете через объект обращаться к полям связанной формы. Для списка это возможность использовать редактируемый список. В Заметках это свойство используется для связи с формой, из которой можно добавлять множество значений.

В качестве значений списка могут быть использованы поля: текст, список, заметка, число, счетчик.

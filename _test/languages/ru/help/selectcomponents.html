Чтобы выделить компонент щелкните по нему мышкой. Вокруг компонента повится рамка с манипуляторами. Вы можете перемещать компонент и изменять его размер с помощью манипуляторов.

Чтобы выделить группу объектов, нажмите кнопку мыши и не отпуская переместите мышь в сторону. Все что попадает в прямоугольную область или пересекается, выделяется. Выделить несколько компонентов можно и по другому. Нажмите клавишу Shift и не отпуская выбирайте поочередно компоненты. Выделенные компоненты не имеют манипуляторов, поэтому для изменения размера можно воспользоваться комбинацией клавиш Shift-Стрелка. С помощью комбинации Ctrl-Стрелка можно перемещать компоненты.

Если компоненты находятся в контейнере то для выделения мышкой нужно зажать клавишу Ctrl.

Если нажать клавишу Esc, то выделится родительский компонент - контейнер.

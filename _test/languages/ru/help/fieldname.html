Имя поля отображается в диалогах, используется в выражениях и шаблонах, отображается в заголовках таблиц. Старайтесь давать полям уникальные имена, чтобы программа могла правильно определить поле. Недопустимо использовать в имени символы: [ ] ' " | { } < > !. В некоторых случаях это может приводить к ошибкам, т. к. эти символы используются в выражениях.

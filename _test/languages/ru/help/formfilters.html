Вы можете создать предварительные настройки фильтра формы для разных случаев. В режиме пользователя, вы можете воспользоваться ими из меню кнопки Фильтр на панели инструментов. При настройке фильтров необязательно вводить значения, можно просто выбрать поля, по которым вы хотите выполнять отбор.

Первый фильтр в списке используется программой по умолчанию при запуске программы или открытии закладки. Если ни одного фильтра не указано, то считываются все записи.

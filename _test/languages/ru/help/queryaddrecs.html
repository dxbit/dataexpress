Если запрос простой, т. е. данные вытягиваются из одного источника без группировок, становится возможным редактировать данные связанной формы. При добавлении записи программа проверяет состав полей фильтра источника и подставляет в соответствующие поля формы значения. Значения подставляются только в те поля, для которых использовалась операция =.

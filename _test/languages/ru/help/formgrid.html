Это свойство позволяет настроить внешний вид табличной части, расположение и размер столбцов, сортировку и пр.

Щелкните по заголовку столбца, чтобы настроить сортировку. Чтобы добавить столбец для сортировки, щелкните по заголовку с зажатой клавишей Ctrl.

Вы можете разрешить/запретить редактирование в таблице. В режиме только чтение можно пользоваться контекстным поиском. Просто начните вводить текст и программа будет искать по текущей колонке. Если дважды щелкнуть по записи, то откроется форма для редактирования записи (если представление Только таблица). Форму редактирования можно также открыть, дважды щелкнув по левой части таблицы, даже если табличная часть редактируемая.

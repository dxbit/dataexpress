Шаблон - это специальным образом подготовленный документ для вывода данных формы на печать. Шаблоны создаются в каком-либо текстовом редакторе, поддерживающий один из форматов: DOCX, DOCM, XML (MS Word), ODT, HTML. Это может быть Microsoft Word, OpenOffice Writer и т. д. Документ состоит из произвольного текста, таблиц, рисунков (документ может иметь любую структуру). Поля формы и вычисляемые поля пишутся в квадратных скобках. Например: [Дата], [Сотрудник|ФИО], [Сумма прописью].

Также в шаблоне используются теги, которые указывают из какой формы брать данные. Теги пишутся в фигурных скобках, их всего четыре: form, grid, group и end. Примеры записи: {form|Ветераны}, {grid|Оказанные услуги}. Тег {end} используется в паре с form, grid и указывает на конец записи. Тег {end} используется в паре с тегом {group} и указывает на конец группы.

Если поля находятся между тегами form-end или grid-end, печатаются все записи формы. Если печатаются записи главной формы, то количество записей зависит от настроек фильтра. Если эти теги не использовать, то печатается только текущая запись.

Тег group испльзуется для группировки данных по какому-либо значению. Например:
{group|Группа услуг|Название}[Группа|Название]{end}

После group идет название поля группировки без квадратных скобок. Это может быть поле формы/запроса или вычисляемое поле.

На самом деле тег group ничего не группирует, а просто выводит данные внутри group-end. Но данные печатаются не каждый раз, а только тогда, когда значение поля группировки меняется. Для корректного вывода данных на печать, данные должны сортироваться по полю группировки (сортировка по этому полю должна осуществляться в первую очередь).

Теги group используется внутри тегов form-end и grid-end, их может быть несколько. Вложенные теги group не допускаются.

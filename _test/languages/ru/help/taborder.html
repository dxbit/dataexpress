Порядок перехода определяет очередность перехода между компонентами по клавише Tab. Кнопка Авто сама настраивает порядок перехода: слева-направо, сверху-вниз. Чтобы воспользоваться этой кнопкой выберите самый верхний компонент - форму.

Когда открывается форма редактирования записи, фокус переходит на самый первый компонент.

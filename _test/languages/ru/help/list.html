Это поле позволяет выбрать значение из списка. Список может быть фиксированным или заполняться из справочника (формы). Не путайте список и объект. Внешне они похожи, но хранят данные по разному. Список хранит текст, а объект хранит идентификатор объекта - число. Если Вы выбрали форму в качестве источника списка, то изменения в связанной форме не будут отражаться на данных списка. Длина текста по умолчанию 150 символов.

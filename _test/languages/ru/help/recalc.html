Эта команда будет полезна, когда вы добавляете новое поле с выражением в рабочую базу данных или меняете что-то в существующем. Будет произведен принудительный расчет поля по записям формы. Поля, которые ссылаются на это поле, тоже будут пересчитаны. Перед использованием команды с помощью фильтра отберите записи, в которых нужно выполнить пересчет.

Выберите из списка основную или подчиненную форму. Выберите поле, в котором необходимо пересчитать выражение. В список попадают только поля с выражениями.

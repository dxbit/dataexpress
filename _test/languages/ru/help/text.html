Используется для хранения текста. Длина текста по умолчанию: 150 символов. Если текст - это адрес сайта или электронной почты, то вы можете перейти по этому адресу, щелкнув по полю с нажатой клавишей Ctrl или выбрав в контекстном меню пункт Перейти по ссылке.

Поле может содержать выражение. Используйте эту возможность, если вам надо отображать составные данные в списке или объекте.

Важно! Пустое текстовое поле хранит не пустую строку, а значение Null. Это надо учитывать при составлении выражений.

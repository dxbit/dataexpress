Эта команда позволяет заменить значение некоторого поля. Перед использованием команды с помощью фильтра отберите записи, в которых необходимо заменить значение поля.

Выберите из списка основную или подчиненную форму. Выберите поле, значение которого нужно заменить. В список не попадают поля с выражениями. В поле Выражение введите выражение. Выражение составляется точно так же, как выражение в поле формы. В выражении можно ссылаться на поля текущей и родительской формы. Выражение должно возвращать значение совместимое с полем. Поля с выражениями, которые ссылаются на выбранное поле, пересчитываются автоматически.

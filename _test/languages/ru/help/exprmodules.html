Модули выражений расширяют возможности выражений за счет добавления новых функций, написанных с помощью встроенного в DataExpress языка программирования RemObject Pascal Script. Эта технология разработана для той категории разработчиков, которые слабо разбираются или вообще не разбираются в программировании, но понимают выражения. Модули выражений представляют собой файл с исходным кодом на языке Pascal Script с расширением *.epas. Нужно просто добавить этот файл в вашу базу данных, чтобы воспользоваться новыми функциями в выражениях. Для этого нажмите кнопку "Добавить" и выберите нужный файл. После добавления программа компилирует имеющиеся модули. Если в процессе компиляции будет обнаружена ошибка, то программа известит об этом. В случае ошибки все имеющиеся модули работать не будут. Нужно либо исправить ошибку в модуле, либо вообще удалить ошибочный модуль из базы.

Модули выражений являются заменой плагинам. По сравнению с плагинами модули имеют несколько преимуществ:
  • В руках разработчика вся мощь скриптов. Разработчик может в виде функции реализовать любое действие, которое только можно сделать с помощью скриптов. Плагины же не имеют доступа к интерфейсу и объектам программы.
  • Простота разработки. Для разработки модулей используется редактор скриптов DataExpress. Для разработки плагинов требуется отдельная среда разработки.
  • Малый размер. Модуль выражений представляет собой простой текстовый файл, который может занимать от килобайта до нескольких десятков килобайт.
  • Интеграция с базой данных. Модуль добавляется прямо в вашу базу данных и загружается клиентами при подключении к базе данных. Файлы плагинов нужно хранить на каждом клиенте.

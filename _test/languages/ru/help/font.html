Шрифт можно настроить почти для всех компонентов. Чтобы задать шрифт сразу нескольким компонентам, выделите их и выберите пункт Шрифт в контекстном меню. При этом автоматически устанавливается шрифт первого выделенного компонента. Это можно использовать, если желаемый шрифт уже задан для какого-либо компонента.

Компоненты наследуют шрифт контейнера (формы, группы, закладки). При создании компонента в контейнере ему автоматически присваивается шрифт контейнера. При изменении шрифта контейнера изменяется и шрифт компонентов. Если шрифт компонента отличается от шрифта контейнера, то изменение шрифта контейнера не влияет на шрифт компонента.
